## 📝 프로젝트 소개

`MiriCanvas Studio`는 웹 포인터 조작만으로 사각형, 원, 텍스트, 사용자 이미지 레이어를 동적으로 생성하고 자유롭게 조작(드래그, 리사이징, 투명도/색상 변경)할 수 있는 **웹 기반 그래픽 에디터 프로젝트**입니다. 

실제 매장 및 일상에서 디지털 편집 툴 사용에 어려움을 겪는 이들을 위한 **배리어 프리(Barrier-free) 인터페이스**를 목표로 설계되었으며, 레이어가 늘어나도 버벅임 없는 **60fps급 렌더링 최적화**를 달성했습니다.

## 💻 요구 사항 및 설치/실행방법

### 1. 요구 사항 (Prerequisites)

이 프로젝트를 클론하여 로컬 환경에서 실행하려면 다음 프로그램이 설치되어 있어야 합니다.

* **Node.js**: `v18.0.0` 이상 권장
* **npm**: `v9.0.0` 이상 권장

### 2. 설치 및 실행 (Installation & Setup)

```bash
# 1. Repository 클론
git clone [https://github.com/puara-g/miricanvas-new.git](https://github.com/puara-g/miricanvas-new.git)

# 2. 프로젝트 디렉토리 이동
cd miricanvas-new

# 3. 의존성 패키지 설치
npm install

# 4. 개발 서버 실행
npm run dev
